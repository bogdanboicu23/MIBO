# Finance Expenses

## Supported handlers

| Handler | Downstream call | Arguments |
| --- | --- | --- |
| `finance.expenses.list` | `GET /api/expenses` | `userId`, `category`, `limit`, `skip`, `sortBy`, `order` |
| `finance.expenses.categories` | `GET /api/expenses/categories` | none |

## Field hints

### `finance.expenses.list`

- `entityType = expense_list`
- `collectionPath = items`
- `labelField = description`
- `valueField = amount`
- `categoryField = category`

### `finance.expenses.categories`

- `entityType = category_list`
- `collectionPath = items`
- `labelField = name`

## Implementation notes

- `finance.expenses.categories` converts a raw array of strings into `{ items: [{ name, slug }] }`
- in the current implementation, `slug` is equal to `name`
