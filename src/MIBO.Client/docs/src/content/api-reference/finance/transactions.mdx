# Finance Transactions

## Supported handlers

| Handler | Downstream call | Arguments |
| --- | --- | --- |
| `finance.transactions.list` | `GET /api/transactions` | `userId`, `type`, `category`, `limit`, `skip`, `sortBy`, `order` |
| `finance.transactions.search` | `GET /api/transactions/search` | `q`, `query`, `limit`, `skip` |

## Field hints

- `entityType = transaction_list`
- `collectionPath = items`
- `labelField = description`
- `valueField = amount`
- `categoryField = category`

## Implementation notes

- search requires `q` or `query`
- Action Service forwards the request to BankService and returns the normalized result to the caller
