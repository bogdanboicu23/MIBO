# Summary API

Get high-level financial summaries for users.

**Base path:** `/api/summary`

<Callout type="info" title="Authentication Required">
  All summary endpoints require a valid JWT Bearer token in the `Authorization` header.
</Callout>

<SectionDivider />

<SectionHeader title="Full Summary" description="Get a comprehensive financial overview for a user" />

<ApiEndpoint
    method="GET"
    path="/api/summary/user/{userId}"
    description="Get a comprehensive financial summary for a user including balance, expenses, and account info."
    params={[
        { name: "userId", type: "number", required: true, description: "User ID" },
    ]}
    response={`{
    "userId": 42,
    "totalBalance": 24500.00,
    "totalExpenses": 3200.00,
    "currency": "USD",
    "accounts": [
        { "id": 1, "name": "Main Checking", "balance": 12450.00 },
        { "id": 2, "name": "Savings", "balance": 12050.00 }
    ],
    "recentTransactions": 45,
    "activeBudgets": 3
}`}
/>

<Callout type="tip" title="Dashboard Data">
  The full summary endpoint is ideal for populating dashboard views, as it aggregates data from accounts, transactions, expenses, and budgets in a single request.
</Callout>

<SectionDivider />
