# Analytics API

Advanced analytics for expense tracking, category breakdowns, and time-series data.

**Base path:** `/api/analytics`

<Callout type="info" title="Authentication Required">
  All analytics endpoints require a valid JWT Bearer token in the `Authorization` header.
</Callout>

<SectionDivider />

<SectionHeader title="Expense Analytics" description="Get expense analytics with optional date range and period grouping" />

<ApiEndpoint
    method="GET"
    path="/api/analytics/expenses/user/{userId}"
    description="Get expense analytics with optional date range and period grouping."
    params={[
        { name: "userId", type: "number", required: true, description: "User ID" },
        { name: "startDate", type: "string", required: false, description: "Start date (ISO 8601)" },
        { name: "endDate", type: "string", required: false, description: "End date (ISO 8601)" },
        { name: "period", type: "string", required: false, description: "Grouping period (see period values below)" },
    ]}
/>

<SectionDivider />

<SectionHeader title="Category Breakdown" description="Get expense breakdown by category with chart-ready data" />

<ApiEndpoint
    method="GET"
    path="/api/analytics/expenses/user/{userId}/categories"
    description="Get expense breakdown by category with chart-ready data."
    params={[
        { name: "userId", type: "number", required: true, description: "User ID" },
        { name: "startDate", type: "string", required: false, description: "Start date" },
        { name: "endDate", type: "string", required: false, description: "End date" },
        { name: "period", type: "string", required: false, description: "Period filter" },
    ]}
    response={`{
    "userId": 42,
    "startDate": "2024-01-01",
    "endDate": "2024-01-31",
    "totalAmount": 3200.00,
    "currency": "USD",
    "categories": [
        { "name": "Groceries", "amount": 850.00, "percentage": 26.5 },
        { "name": "Dining", "amount": 450.00, "percentage": 14.1 },
        { "name": "Transport", "amount": 200.00, "percentage": 6.25 }
    ],
    "chartData": {
        "labels": ["Groceries", "Dining", "Transport"],
        "values": [850, 450, 200]
    }
}`}
/>

<Callout type="tip" title="Chart-Ready Data">
  The `chartData` field in category and time-series responses provides pre-formatted labels and values that can be directly fed into charting libraries.
</Callout>
