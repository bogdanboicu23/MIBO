# Finance Analytics

## Supported handlers

| Handler | Downstream call | Arguments |
| --- | --- | --- |
| `finance.analytics.expenses` | `GET /api/analytics/expenses/user/{userId}` | `userId`, `period` |
| `finance.analytics.transactions` | `GET /api/analytics/transactions/user/{userId}` | `userId`, `period` |

## Field hints

### `finance.analytics.expenses`

- `entityType = expense_analytics`
- `collectionPath = byCategory`
- `labelField = category`
- `valueField = amount`

### `finance.analytics.transactions`

- `entityType = transaction_analytics`
- `labelField = userId`
- `valueField = netFlow`
