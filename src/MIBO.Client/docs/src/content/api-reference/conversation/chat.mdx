# Chat API

The primary endpoint for sending messages to the MIBO AI assistant.

**Base path:** `/api/v1/chat`

<Callout type="info" title="Authentication Required">
  This endpoint requires a valid JWT Bearer token in the `Authorization` header.
</Callout>

<SectionDivider />

<SectionHeader title="Send Message" description="Send a message to the AI assistant and receive a response with optional generative UI" />

<ApiEndpoint
    method="POST"
    path="/api/v1/chat"
    description="Send a message to the AI assistant and receive a response with text and optional generative UI components."
    body={`{
    "conversationId": "conv-123-uuid",
    "userId": "user-456-uuid",
    "prompt": "Show me my spending this month"
}`}
    response={`{
    "text": "Here's your spending summary for this month. You've spent $1,234.56 across 45 transactions.",
    "uiV1": {
        "type": "expense-chart",
        "data": {
            "total": 1234.56,
            "categories": [
                { "name": "Groceries", "amount": 450.00 },
                { "name": "Dining", "amount": 280.00 }
            ]
        }
    },
    "correlationId": "corr-789-uuid"
}`}
    params={[
        { name: "conversationId", type: "string", required: true, description: "UUID of the conversation" },
        { name: "userId", type: "string", required: true, description: "UUID of the authenticated user" },
        { name: "prompt", type: "string", required: true, description: "The user's message text" },
    ]}
/>

<SectionDivider />

<SectionHeader title="Response Structure" description="Understanding the fields returned by the chat endpoint" />

<FeatureGrid columns={3}>
  <FeatureCard title="text" description="Natural language response from the AI assistant" icon="📝" />
  <FeatureCard title="uiV1" description="Optional generative UI specification for rich interactive components" icon="🎨" />
  <FeatureCard title="correlationId" description="Unique ID for tracking and correlating this interaction" icon="🔗" />
</FeatureGrid>

<SectionDivider />

<SectionHeader title="Generative UI" description="Dynamic UI components rendered from AI responses" />

The `uiV1` field contains a dynamic UI specification that the client renders. This allows the AI to present rich, interactive components based on the conversation context -- charts, tables, cards, forms, and more.

<Callout type="tip" title="Full Orchestration Pipeline">
  The chat endpoint handles the full orchestration pipeline: planning, tool execution, UI composition, and text generation. For streaming responses, use the [streaming endpoint](/api-reference/conversation/streaming) instead.
</Callout>
