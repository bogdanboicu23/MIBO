# Chat System

MIBO's chat system provides a natural language interface for users to interact with their financial data. The system orchestrates AI responses, executes financial operations, and delivers generative UI components in real-time.

<SectionDivider />

<SectionHeader title="How It Works" description="The complete message lifecycle from user input to rendered response" />

<Steps>
  <Step title="User sends a message">
    The user types a prompt via the React client interface.
  </Step>
  <Step title="Chat Orchestrator receives the request">
    The orchestrator coordinates the full response pipeline.
  </Step>
  <Step title="Planner analyzes intent">
    The LLM integration analyzes the intent and selects appropriate tools.
  </Step>
  <Step title="Tool Executor runs selected tools">
    Financial queries, account operations, and other backend calls are executed.
  </Step>
  <Step title="UI Composer generates dynamic UI">
    Dynamic UI specifications are created based on the returned data.
  </Step>
  <Step title="Text Composer produces a response">
    A human-readable natural language response is generated.
  </Step>
  <Step title="Response streams back to the client">
    The combined response is delivered via SSE to the React client.
  </Step>
</Steps>

<SectionHeader title="Example Interactions" description="How the system handles common user queries" />

<Callout type="tip" title="Spending Query">
**User:** "Show me my spending this month"
- Planner selects the expense analytics tool
- Tool Executor queries the Finance Data Service
- UI Composer generates a spending chart component
- Text Composer writes a summary
</Callout>

<Callout type="tip" title="Balance Query">
**User:** "What's my account balance?"
- Planner selects the account summary tool
- Tool Executor fetches account data
- UI Composer renders an account card
- Text Composer provides the balance information
</Callout>
