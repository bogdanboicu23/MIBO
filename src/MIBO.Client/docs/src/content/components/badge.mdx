# Badge

An inline badge component for labels, statuses, and tags. Badges are small, non-interactive elements that convey quick visual information such as status indicators, category labels, or count tags throughout the MIBO interface.

<Callout type="tip" title="Variant Selection">
  Use `default` for high-emphasis labels, `subtle` for softer status indicators, and `outline` for minimal, bordered tags that blend with surrounding content.
</Callout>

<SectionDivider />

<SectionHeader title="Usage" description="How to import and use the Badge component" />

```tsx
import { Badge } from "@/components/ui";

<Badge variant="subtle">Active</Badge>
<Badge variant="outline">Pending</Badge>
```

<Callout type="info" title="Composability">
  Badges work well inside Table cells, Card headers, and alongside text. They render as inline `span` elements, so they flow naturally within text content.
</Callout>
