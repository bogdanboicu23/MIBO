# Component Library

MIBO includes a purpose-built component library using **React 19**, **TypeScript**, and **Tailwind CSS v4**. All components follow a consistent design language with the zinc color palette and support both light and dark modes.

<SectionHeader title="Usage Pattern" description="How to import and use components in your project" />

All components use the `cn()` utility for conditional class merging:

```tsx
import { Button } from "@/components/ui";

function MyComponent() {
    return (
        <Button variant="solid" size="lg" onClick={handleClick}>
            Get Started
        </Button>
    );
}
```

<Callout type="tip" title="Import Path">
  All components are exported from the barrel file at `@/components/ui`. You can import any component from this single path.
</Callout>
