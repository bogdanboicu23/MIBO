# Component Library

This section documents the sandbox UI registry used by the main React client to render assistant-generated UI.

## What this section covers

- runtime registry components used by the generative UI system
- component-specific props and usage guidance
- the relationship between data sources, actions, and rendered components

## Scope note

This is not a generic design-system catalog. It is documentation for the components that participate in the **runtime UI composition model** used by assistant responses.
