# Input

A form input component with built-in label, hint text, and error state support. The Input component handles common form patterns like validation feedback and helper text, so you can build forms quickly without extra wrapper components.

<Callout type="info" title="Auto-Generated IDs">
  When no `id` prop is provided, the Input component automatically generates a unique ID to link the label and input elements for proper accessibility.
</Callout>

<SectionHeader title="Usage" description="How to import and use the Input component in forms" />

```tsx
import { Input } from "@/components/ui";

function LoginForm() {
    return (
        <form>
            <Input label="Email" type="email" placeholder="you@example.com" />
            <Input label="Password" type="password" error={errors.password} />
        </form>
    );
}
```

<Callout type="warning" title="Error State">
  When the `error` prop is provided, it replaces the `hint` text and changes the input border to red. Make sure your error messages are clear and actionable.
</Callout>
