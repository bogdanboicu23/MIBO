# IconButton

A fixed-size square button designed for icon-only actions. The IconButton provides a consistent hit target and hover state for toolbar actions, close buttons, settings toggles, and other icon-driven interactions throughout the MIBO interface.

<SectionHeader title="Usage Guidelines" description="Best practices for icon-only buttons" />

<Callout type="warning" title="Always Provide aria-label">
  Since IconButton has no visible text, you must always provide an `aria-label` prop to describe the action for screen readers. Without it, the button is inaccessible to assistive technology users.
</Callout>

<Callout type="tip" title="Icon Sizing">
  Use `h-5 w-5` for SVG icons inside the IconButton for the best visual balance with the default button size. The button provides its own padding around the icon content.
</Callout>
