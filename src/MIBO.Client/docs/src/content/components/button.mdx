# Button

A flexible button component with four visual variants and three sizes. The Button component is the primary interactive element in the MIBO UI library, supporting solid, outline, secondary, and ghost styles to cover every use case from primary actions to subtle triggers.

<Callout type="tip" title="Size Guidelines">
  Use `sm` for inline or compact layouts, `md` for standard forms and actions, and `lg` for hero sections or primary call-to-action areas.
</Callout>

<SectionHeader title="Usage" description="How to import and use the Button component" />

```tsx
import { Button } from "@/components/ui";

function Example() {
    return (
        <Button variant="solid" size="lg" onClick={() => alert("Clicked!")}>
            Submit
        </Button>
    );
}
```

<Callout type="info" title="Accessibility">
  The Button component forwards all native button HTML attributes, including `disabled`, `type`, and `aria-*` props. Always provide meaningful text content or an `aria-label` for screen readers.
</Callout>
