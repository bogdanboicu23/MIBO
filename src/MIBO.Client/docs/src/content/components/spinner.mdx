# Spinner

A simple animated loading spinner using CSS border animation. The Spinner component provides visual feedback during async operations like data fetching, form submission, or content loading. It is lightweight and customizable through standard Tailwind utility classes.

<Callout type="tip" title="Size and Color">
  Customize the spinner size with `h-*` and `w-*` classes, and color with `text-*` classes. The default size is `h-4 w-4` with the current text color.
</Callout>

<Callout type="info" title="Accessibility">
  The Spinner renders as a `span` element with a CSS border animation. For screen readers, pair it with visually-hidden loading text or an `aria-busy` attribute on the parent container.
</Callout>
