# Select

A styled select dropdown component following the same pattern as Input. The Select component provides a consistent form experience with label, hint text, and error state support, making it easy to build validated forms with dropdowns.

<Callout type="tip" title="Consistent Form API">
  The Select component shares the same `label`, `hint`, and `error` API as the Input component. This makes it easy to build forms with a mix of inputs and dropdowns that look and behave consistently.
</Callout>

<SectionDivider />

<SectionHeader title="Usage" description="How to import and use the Select component" />

```tsx
import { Select } from "@/components/ui";

<Select label="Account Type" onChange={handleChange}>
    <option value="checking">Checking</option>
    <option value="savings">Savings</option>
</Select>
```

<Callout type="info" title="Native Options">
  The Select component uses native `<option>` elements as children. This ensures full browser compatibility and accessibility support without requiring a custom dropdown implementation.
</Callout>
