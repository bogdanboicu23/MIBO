# Textarea

A minimal textarea component designed to be embedded within card or form containers. The Textarea features a transparent background and no resize handle by default, making it ideal for chat interfaces and form layouts where the parent container provides the visual boundary.

<Callout type="info" title="Transparent Background">
  The Textarea has a transparent background and no resize by default. It's designed to be placed inside a styled container like a Card.
</Callout>

<Callout type="tip" title="Chat Interface Pattern">
  For chat-style UIs, wrap the Textarea in a Card with a send button in the CardFooter. This creates a clean message input area that matches the MIBO design language.
