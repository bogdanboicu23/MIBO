# Sandbox Registry Components

This section documents the components used by the client-side sandbox registry. These are the render targets used by `ui.v1` payloads built from assistant responses.

## Documented registry areas

- action and workflow components
- commerce and product components
- tabular and charting components
- markdown and general content components
- music and productivity components present in the runtime registry

## Runtime note

The registry in the main client includes a few components that are newer than the older docs set, including:

- Pomodoro timer rendering
- Spotify player rendering
- Spotify track-list rendering

Those are included here so the docs reflect the actual client registry.
