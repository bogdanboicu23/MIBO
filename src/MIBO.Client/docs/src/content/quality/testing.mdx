# Testing

The main automated test entry point is:

```bash
dotnet test tests/MIBO.Tests.sln
```

## CI test workflow

`.github/workflows/tests.yml` currently:

- restores `tests/MIBO.Tests.sln`
- builds the solution in `Release`
- runs `dotnet test`
